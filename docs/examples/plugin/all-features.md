# 全功能演示

集成 GE 几乎全部插件，展示综合能力：12 内置 Shape、A\* 避障、自环、流动虚线、阴影、四向 Port。

<ExamplePreview src="plugin/all-features" :height="560" />

## 快捷键

- `Delete` 删除选中（含其连线）
- `Ctrl/Cmd + Z` 撤销 · `Ctrl/Cmd + Shift + Z` 重做
- `Ctrl/Cmd + A` 全选 · `Ctrl/Cmd + C / V` 复制粘贴 · `Ctrl/Cmd + D` 克隆
- `Alt + 拖拽节点` 拖出创建新连线
