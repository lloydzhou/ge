# `ge-core` 公开运行时接口测试覆盖静态审计

> 状态更新（2026-07-16）：本轮补强已完成。以下发现 A–Q 中的中严重度与低到中严重度缺口均已关闭，并补充了相关低严重度基础契约。验证结果：全量 10 个测试文件、121 个测试通过；`@antv/ge-core` 构建通过；`tsc --noEmit` 通过。
>
> 最终边界契约：
> - `bboxFromPoints([])` 返回 `{ x: 0, y: 0, width: 0, height: 0 }`。
> - `manhattanAStarRouter` 对非有限、零或负分辨率，以及会造成过大障碍网格的极小分辨率，立即降级到 `manhattanRouter`。
> - 网格布局的 `cols` 必须是正整数；力导向布局拒绝非法迭代次数、宽高和理想距离；圆形布局拒绝非有限参数。
> - 层次布局和树布局忽略引用未知节点的边，输出只包含输入节点。
> - 边锚点对空路径、单点、非法比例和索引采用有限坐标降级；连接器的非有限半径或张力回退默认值。
